# Automotive-Sales-Performance-Market-Analysis
A comprehensive analysis of an automotive group's sales performance by IBM Cognos Analysis, focusing on regional trends, model popularity, and revenue growth. This project bridges the gap between raw sales data and executive decision-making.
# Tools & Skills Demonstrated:

Data Visualization: Created professional slide decks to translate complex data into actionable insights.

Business Intelligence: Developed a data module for standardized automotive reporting.

Strategic Planning: Provided recommendations on inventory focus based on sales performance data.

# Key Analytical Areas:

Sales Trend Analysis: Identified seasonal fluctuations and year-over-year growth patterns in car sales.

Inventory Performance: Analyzed which models (e.g., SUVs vs. Sedans) had the highest turnover rates.

Regional Benchmarking: Evaluated performance across different dealership locations to identify underperforming markets.

Data Modeling: Documented a structured data module to ensure consistent reporting across the auto group.
